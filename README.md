# Install Instructions:
- Download the .Zip file
- Right Click It And hit "Extract"
- Open your browser:
  * IF ON CHROME -- Go to chrome://extensions
  * IF ON EDGE -- Go to edge://extensions
- Enable Developer Mode
- Click "Load..."
- Open the extracted folder
- Enjoy
